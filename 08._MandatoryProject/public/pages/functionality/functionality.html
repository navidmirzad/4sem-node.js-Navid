What is Express?

NodeJS is a open source runtime for JavaScript. Additionally we have used the Express whichs is a framework.

  • Express is a popular web framework, used for web applications.
  • Great at creating robust APIs with minimal effort thanks to its routing.

Since we use Node.JS as our backend runtime enviroment to allow us write server-side code using JavaScript

Nodemon is a great tool, we install by writing "npm install -g nodemon" which adds it to our package.json file.
Nodemon allows us to detect changes in our code base. So if there is an error in our code, it even lets us know where.

Nodemon is also customizeable, for example in our nodemon.json we told it listen to js, json, & html
