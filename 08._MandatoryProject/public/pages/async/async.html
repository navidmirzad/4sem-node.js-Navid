What is Async, await and promises?

Like I showed earlier in /fetch a promise is an asynchronous event in JavaScript.
It has 3 states:
  • Pending: initial state before the event has happend.
  • Resolved: After the operation completed succesfully.
  • Rejected: If there was an error during execution, the promise gets rejected.

Key differences between .then & await/async

  • Error handling if using .then, is writing the 
    catch((error) => console.log(error));
  • Error handling if using async/await: 
    try { relevant code } catch (error) { console.log(error); }

Async/Await is typically prefered over the .then method as it provides a cleaner and more customizeable way of working with promises.
