How to start a Node.JS, Express, Nodemon from bash

#prerequisites: 

  • Node.JS installed on your computer
  • npm installed if not / 
    npm install

#1 Initialize the node project:
npm init

#2 Download Express Node framework: 
npm install express

#3 Install Nodemon which is a tool that restarts our application everytime it detects a change, instead of using have to restart. 
npm install -g nodemon // Global download
npm install --save-dev nodemon // local download
